# Política de Segurança — PortfolioHUB

## Práticas de Segurança Adotadas

- Autenticação de dois fatores (2FA) ativada na conta GitHub
- Branch `main` protegida: alterações somente via Pull Request com aprovação
- Variáveis sensíveis armazenadas como **GitHub Secrets** (nunca no código)
- Dependabot ativado para alertas automáticos de vulnerabilidades
- HTTPS obrigatório no GitHub Pages

## Reportando uma Vulnerabilidade

Se você encontrar alguma vulnerabilidade de segurança neste projeto:

1. **Não abra uma Issue pública**
2. Entre em contato diretamente pelo e-mail do responsável pelo repositório
3. Descreva o problema com o máximo de detalhes possível
4. Aguarde o retorno em até **7 dias úteis**
